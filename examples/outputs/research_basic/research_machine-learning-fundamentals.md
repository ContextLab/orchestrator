### 1. **Definition and Overview**
- **Machine Learning (ML)** is a subset of artificial intelligence that enables systems to learn from data, identify patterns, and make decisions with minimal human intervention.
- ML automates analytical model building using algorithms that iteratively learn from data.

---

### 2. **Core Concepts**
- **Data:** The foundation of ML; models require large volumes of quality data for training.
- **Algorithms:** Mathematical procedures or formulas that process data and learn patterns (e.g., linear regression, decision trees, neural networks).
- **Model Training:** The process where an algorithm finds patterns in data and optimizes parameters to make accurate predictions.
- **Prediction/Inference:** After training, the model can predict outcomes or classify new, unseen data.

---

### 3. **Types of Machine Learning**
- **Supervised Learning:** The model is trained on labeled data (input-output pairs). Example: Spam detection in emails.
- **Unsupervised Learning:** The model explores input data without labeled responses, learning from the inherent structure (e.g., clustering customers by purchase behavior).
- **Reinforcement Learning:** The model learns by interacting with an environment, receiving feedback (rewards or penalties) for actions.

---

1. **Problem Framing:** Define the problem and the type of learning required.

4. **Evaluation:** Test model performance using metrics like accuracy, precision, recall, etc.
